# BindWidgets

Using JS to create UI in raylib using MUjs interpreter
> Inspired by [Lust2d](https://github.com/tsoding/lust2d).

# Compile

This project use Nob for compiling, just do:
```
cc -o nob nob.c
./nob
```
The use for nob is to always be possible to compile this project only with a C compiler no requiring external dependencies

# usage
Run bwidgets with a JS file
```
./build/bwidgets <Main.js>
```
